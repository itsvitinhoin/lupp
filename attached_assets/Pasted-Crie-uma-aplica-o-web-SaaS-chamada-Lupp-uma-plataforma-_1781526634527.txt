Crie uma aplicação web SaaS chamada Lupp, uma plataforma de Video Commerce para e-commerces.

A Lupp transforma lojas online em uma experiência de descoberta por vídeos verticais, no estilo TikTok/Reels, com vídeos compráveis, produtos linkados, feed vertical, widgets para páginas de produto, métricas de performance e personalização visual.

O objetivo agora é criar somente a interface completa e navegável, com dados mockados, estados visuais e componentes bem estruturados para que posteriormente outro agente implemente todas as funcionalidades reais, backend, banco de dados, autenticação, integrações e APIs.

Não implemente backend real. Não conecte APIs reais. Não faça upload real. Use dados simulados, componentes reutilizáveis e arquitetura limpa.

⸻

1. Stack desejada

Use:

* React
* TypeScript
* Vite
* Tailwind CSS
* shadcn/ui ou componentes próprios bem organizados
* lucide-react para ícones
* recharts para gráficos
* framer-motion para animações suaves, se disponível
* layout responsivo desktop/mobile
* código limpo, modular e preparado para futura implementação real

A aplicação deve parecer um SaaS moderno, premium, tecnológico, dark mode first e mobile-first.
